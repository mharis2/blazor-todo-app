# Blazor Todo List

A simple Todo List application built with Blazor for learning purposes.

## Description

This project is a basic Todo List application that allows users to add, delete, and mark tasks as complete. It's built using Blazor, a modern web framework from Microsoft. The main aim of this project was to understand the fundamentals of Blazor and how it can be used to build interactive web applications.

## Usage

### How to Use the Project

The Todo List application is straightforward:

1. Enter a task in the input field.
2. Click the "Add" button to add the task to the list.
3. Tasks can be marked as complete by checking the checkbox next to them.
4. Tasks can be deleted by clicking the "Delete" button next to them.

**Note:** This project was mainly for learning purposes and does not require any authentication.
